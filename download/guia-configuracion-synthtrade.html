A Opcion A: Usar el Bot desde el Navegador

Esta es la forma mas rapida de comenzar. El bot ya esta alojado y funcionando. Solo necesitas abrirlo en tu navegador, conectar tu token de API de Deriv y comenzar a operar.

Paso 1: Crear Cuenta en Deriv

① Registrarse en Deriv

Abre tu navegador y ve a https://app.deriv.com

Haz clic en "Sign up" y completa el formulario de registro con tu correo electronico.

Verifica tu correo electronico siguiendo el enlace que Deriv te enviara.

② Usar Cuenta DEMO (Recomendado)

Despues de registrarte, Deriv te otorga automaticamente una cuenta demo con $10,000 USD virtuales.

Es MUY recomendable que pruebes el bot primero con la cuenta demo antes de usar dinero real. Puedes cambiar entre cuenta demo y real en cualquier momento desde el menu de Deriv.

⚠

Importante: Nunca uses dinero que no puedas permitirte perder. El trading de indices sinteticos conlleva un alto riesgo. Siempre prueba tus estrategias con la cuenta demo primero.

Paso 2: Obtener el Token de API

③ Generar Token de API

Desde tu cuenta de Deriv, ve a Settings > API Token (o ve directamente a https://app.deriv.com/account/api-token).

Haz clic en "Create" para crear un nuevo token.

Asigna un nombre al token (por ejemplo: "SynthTrade Bot").

Selecciona los siguientes scopes (permisos) obligatoriamente:

Scope Descripcion Necesario
Trade Permite al bot comprar y vender contratos Si
Payments Permite consultar balance y transacciones Si
Admin Acceso completo a funciones administrativas Si
Read Permite leer datos de la cuenta Recomendado
④ Copiar y Guardar el Token

Haz clic en "Create token". Deriv generara una cadena de texto larga (el token).

Copia el token y guardalo en un lugar seguro. Lo necesitaras para conectar el bot. Este token es como una contrasena: NO lo compartas con nadie.

🔒

Seguridad: El token de API da acceso completo a tu cuenta de trading. Guardalo como guardarias cualquier contrasena importante. Si crees que alguien tiene tu token, eliminalo inmediatamente desde la seccion de API Token en Deriv y genera uno nuevo.

Paso 3: Conectar el Bot

⑤ Abrir SynthTrade Pro

Abre el enlace del bot en tu navegador (Chrome, Firefox, Edge, etc.). El bot carga una interfaz profesional tipo terminal de trading.

⑥ Pegar el Token de API

En el panel izquierdo, encontraras la seccion "Connection".

Haz clic en el campo de texto que dice "Enter Deriv API Token".

Pega tu token de API (usando Ctrl+V o clic derecho > Pegar).

Haz clic en el boton "Connect" (verde).

✔

Listo! Cuando veas el indicador verde "Live" en la barra superior, significa que estas conectado a Deriv. Veras tu balance de cuenta demo reflejado en la pantalla.

Paso 4: Comenzar a Operar

⑦ Seleccionar Mercado

En el panel izquierdo, bajo "Synthetic Markets", selecciona un indice sintetico. Recomendamos comenzar con Volatility 10 (R_10) ya que tiene menor volatilidad.

⑧ Activar Estrategia

En el panel de "Strategy Engine", activa al menos una estrategia. La estrategia RSI es la mas recomendada para principiantes.

Activa el interruptor de "Auto Trading" si quieres que el bot opere automaticamente, o usa los botones CALL/PUT manualmente.

5 Estrategias Disponibles

El bot incluye 4 estrategias profesionales de trading que pueden combinarse entre si. Cuando multiples estrategias estan activas, el bot usa un sistema de consenso: solo opera cuando la mayoria de estrategias coinciden en la misma direccion.

📈 RSI (Relative Strength Index)

Calcula el RSI con periodo 14. Compra cuando el RSI baja de 30 (sobreventa) y vende cuando sube de 70 (sobrecompra). Es ideal para mercados con rangos definidos y para detectar reversiones de precio. Es la estrategia mas estable y recomendada para principiantes.

📉 MA Crossover (Cruce de Medias Moviles)

Usa dos medias moviles: rapida (5 periodos) y lenta (20 periodos). Genera senal de compra cuando la MA rapida cruza por encima de la MA lenta (Golden Cross), y senal de venta cuando cruza por debajo (Death Cross). Excelente para identificar tendencias de mediano plazo.

📊 Bollinger Bands (Bandas de Bollinger)

Calcula bandas superior e inferior con 2 desviaciones estandar sobre una media movil de 20 periodos. Compra cuando el precio toca la banda inferior (posible rebote) y vende cuando toca la banda superior (posible retroceso). Efectivo en mercados con volatilidad variable.

⚡ Spike Detection (Deteccion de Espigas)

Detecta movimientos bruscos de precio que superan 2.5 veces la desviacion estandar. Especialmente disenado para indices Boom y Crash, donde los saltos de precio son frequentes. Opera en la direccion del impulso del precio. Esta estrategia es la mas agresiva.

💡

Tip: Para mejores resultados, activa RSI + Bollinger Bands. Estas dos estrategias se complementan bien: RSI detecta sobrecompra/sobreventa mientras que Bollinger Bands confirma los extremos de precio con soporte estadistico.

B Opcion B: Instalar en tu Laptop

Si prefieres ejecutar el bot directamente en tu computadora, sigue estos pasos. Necesitaras conocimientos basicos de uso de terminal/linea de comandos.

Herramientas Necesarias

💻 Node.js (v18+)

Entorno de ejecucion de JavaScript. Descarga desde nodejs.org

🔌 Git

Sistema de control de versiones. Descarga desde git-scm.com

Paso 1: Instalar Node.js

① Descargar e Instalar Node.js

Ve a https://nodejs.org y descarga la version LTS (Long Term Support) para tu sistema operativo.

Ejecuta el instalador y sigue las instrucciones por defecto.

Verifica la instalacion abriendo una terminal y ejecutando:

Terminal node --version
npm --version

Deberias ver numeros de version como v18.x.x o v20.x.x. Si aparece un error, reinicia tu computadora e intenta de nuevo.

Paso 2: Instalar Bun (Runtime de JavaScript)

② Instalar Bun

Bun es un runtime de JavaScript ultrarapido. Abre una terminal y ejecuta:

Terminal (Windows - PowerShell) powershell -c "irm bun.sh/install.ps1 | iex"
Terminal (macOS / Linux) curl -fsSL https://bun.sh/install | bash

Verifica con bun --version. Deberia mostrar la version instalada.

Paso 3: Obtener el Codigo del Bot

③ Clonar o Descargar el Proyecto

Si tienes acceso al repositorio Git, clona el proyecto:

Terminal git clone <URL_DEL_REPOSITORIO>
cd synthtrade-pro

Alternativamente, si recibiste los archivos comprimidos, descomprime la carpeta y abrela en tu terminal.

Paso 4: Instalar Dependencias

④ Ejecutar la Instalacion

Dentro de la carpeta del proyecto, ejecuta:

Terminal bun install

Este comando descargara todas las dependencias necesarias (React, Next.js, shadcn/ui, Prisma, recharts, etc.). Puede tardar 1-2 minutos dependiendo de tu conexion a internet.

Paso 5: Configurar la Base de Datos

⑤ Sincronizar la Base de Datos SQLite

El bot usa SQLite (almacenamiento local) para guardar el historial de operaciones:

Terminal bun run db:push

Este comando crea automaticamente la base de datos local con las tablas necesarias.

⑥ Ejecutar el Servidor de Desarrollo

7 Consejos para Operar con Exito

🎯 Comienza Siempre con Cuenta Demo

Usa la cuenta demo durante al menos 1-2 semanas antes de considerar dinero real. Evalua el rendimiento del bot, ajusta las estrategias y familiarizate con la interfaz sin riesgo financiero. Un buen indicador es mantener un win rate superior al 55% de forma consistente en demo antes de pasar a real.

📈 Usa Multiples Estrategias

Activar 2-3 estrategias simultaneamente mejora la fiabilidad. El sistema de consenso solo opera cuando la mayoria de estrategias coinciden, reduciendo falsas senales. La combinacion RSI + Bollinger Bands es especialmente efectiva para indices de volatilidad.

💰 Empieza con Stakes Pequenos

Comienza con $0.35 - $1.00 por operacion, incluso en cuenta demo. Esto te permite tomar mas operaciones, acumular mas datos estadisticos y ajustar la configuracion sin agotar tu capital rapido. Aumenta gradualmente solo cuando tu confianza en la configuracion sea alta.

⏱ Configura los Limites de Riesgo

Nunca operes sin limites de riesgo. Configura un maximo de perdida diaria ($10-20 para empezar), limite de trades por sesion (20-30), y detencion por perdidas consecutivas (3-5). Estos parametros evitan que una mala racha destruya tu capital.

📊 Desactiva Martingale al Principio

El sistema Martingale (duplicar la apuesta tras cada perdida) puede generar ganancias rapidas pero tambien perdidas masivas. Mantenlo desactivado hasta que tengas al menos 2 semanas de datos positivos consistentes en cuenta demo.

🔔 Activa Sonidos y Notificaciones

En el panel de Risk Management, activa los efectos de sonido y las notificaciones del navegador. Asi podras escuchar cuando se abren o cierran operaciones sin estar mirando la pantalla constantemente.

8 Solucion de Problemas

No puedo conectarme a Deriv

  • Verifica el token: Asegurate de que copiaste el token completo sin espacios extras al inicio o final.
  • Scopes correctos: El token debe tener los scopes Trade, Payments y Admin habilitados.
  • Token valido: Si eliminaste y recreaste el token, usa el nuevo. Los tokens antiguos se invalidan.
  • Conexion a internet: Verifica que tu conexion este activa y no este bloqueada por un firewall.

El grafico no muestra datos

  • Selecciona un mercado: Debes seleccionar al menos un indice sintetico para recibir datos de precio.
  • Espera la carga: Los datos historicos pueden tardar unos segundos en cargar.
  • Reconecta: Si el problema persiste, desconecta y vuelve a conectar.

El auto-trading no abre operaciones

  • Confianza minima: El bot requiere al menos 60% de confianza por defecto para operar.
  • Cooldown: Despues de cada operacion, hay un periodo de espera antes de la siguiente.
  • Sesion pausada: Revisa si la sesion fue pausada por alcanzar un limite de riesgo.
  • Minimo de ticks: El bot necesita al menos 50 ticks de datos antes de analizar.
  • Estrategias activas: Asegurate de tener al menos una estrategia activada.

Error al abrir operaciones

  • Balance insuficiente: Tu stake no puede ser mayor a tu balance disponible.
  • Monto minimo: Deriv requiere un minimo de $0.35 por operacion.
  • Token expirado: Si tu sesion expiro, desconecta y vuelve a conectar con tu token.
